# Netoflix Mini Project

A Netflix-style web app built using **Flask**, integrated with the **YouTube Search API**, featuring dynamic search, click-to-play videos, and a responsive UI. This project demonstrates full-stack development skills, combining backend Python logic with interactive front-end design.

---

## Features

- Dynamic **YouTube search** with fallback **backup data** to ensure content is always available.
- Categorized **Movies** and **TV Shows** sections.
- **Click-to-play video cards** with overlay effects.
- **Horizontal scrollable rows** for easy browsing.
- Responsive design that works on desktop and mobile.
- Backup video catalog ensures the site never appears empty, even if YouTube API fails.

---

## Tech Stack

- **Backend:** Python, Flask
- **Frontend:** HTML, CSS, JavaScript
- **API:** YouTube Search (via `youtubesearchpython` library)
- **Tools:** Git, GitHub
